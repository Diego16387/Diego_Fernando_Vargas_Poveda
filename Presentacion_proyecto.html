Breve explicación del dataset:¶

La base escogida se denomina Facebook Comment Volume Dataset, disponible en el siguiente link:

Ahora bien, Facebook es la principal red social que existe en el mundo. Una red de vnculos virtuales, cuyo principal objetivo es dar un soporte para producir y compartir contenidos. Los comentarios insertados permiten incluir comentarios en publicaciones pblicas (de una pgina o una persona en Facebook) en el contenido de un sitio o una pgina web con gran facilidad. Solo pueden insertarse comentarios pblicos de perfiles y pginas de Facebook.

Caractersticas de la base de datos

La cantidad de datos que se cargan en los servicios de redes sociales est aumentando da a da. Por lo tanto, es posible estudiar los comportamientos dinmicos de los usuarios hacia estos servicios. Con esta base de datos se busca modelar patrones de comentarios del usuario, sobre las publicaciones en las pginas de Facebook y predecir cuntos comentarios se espera que reciba una publicacin en la prxima hora.

Variables

Esta base de datos tiene 54 variables incluyendo la variable de inters (No. de comentarios que puede recibir un post), dentro de las variables a destacar se tiene:

  • No. de comentarios
  • Publicacin promocionada (0,1)
  • Da de la publicacin
  • Horas de medicin (24-48-72)
  • No de visitias a la pgina que tiene la publicacin
  • Clasificacin de la pgina que hace la publicacin
  • Caractersticas tcnicas de la pgina
  • Tamao del post

rea que pertenece los datos

Los datos pertenecen a redes sociales.

import matplotlib.pyplot as plt
from matplotlib import cm
from math import log10


labels = tab4['label']
data = tab4['counts']
#number of data points
n = len(data)
#find max value for full ring
k = 10 ** int(log10(max(data)))
m = k * (1 + max(data) // k)


#radius of donut chart
r = 1.5
#calculate width of each ring
w = r / n 


#create colors along a chosen colormap
colors = [cm.terrain(i / n) for i in range(n)]

#create figure, axis
fig, ax = plt.subplots()
ax.axis("equal")

#create rings of donut chart
for i in range(n):
    #hide labels in segments with textprops: alpha = 0 - transparent, alpha = 1 - visible
    innerring, _ = ax.pie([m - data[i], 
                           data[i]], 
                          radius = r - i * w, 
                          startangle = 90, 
                          labels = ["", labels[i]], 
                          labeldistance = 1 - 1 / (1.5 * (n - i)), 
                          textprops = {"alpha": 0}, 
                          colors = ["white", colors[i]])
    plt.setp(innerring, 
             width = w, 
             edgecolor = "white")
    radius = r - i * w
    plt.text(0, radius - w / 2, f'{labels[i]}  {data[i]} ', ha='right', va='center')
    
#plt.legend()
plt.show()
